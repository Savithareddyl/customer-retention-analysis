# customer-retention-analysis
Customer Retention Analytics Dashboard using Power BI

## Project Overview
This project analyzes customer demographics, transactions, and loyalty program data to understand customer behavior and improve retention.

## Tools Used
- Power BI
- Microsoft Excel
- SQL

## Key Insights
- Identified customer segments with high churn risk
- Analyzed purchasing patterns and loyalty behavior
- Created an interactive dashboard for business decision-making

## Outcome
The dashboard helps businesses identify factors influencing customer retention and improve customer engagement strategies.
